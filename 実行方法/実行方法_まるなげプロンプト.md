最初に時刻を取得

入力は YouTube URL のみです。字幕テキストは渡しません。
あなたが yt-dlp で字幕を取得し、以下の手順で高精度要約してください。

URL: {{　https://www.youtube.com/watch?v=koBLOf-53_g　}}

================================================================================
【字幕取得】yt-dlp を必須で使う
================================================================================

- 動画本体はダウンロードしない（--skip-download）
- 優先: 日本語手動字幕。なければ日本語自動字幕
- 作業用ディレクトリで次を実行する（必要なら yt-dlp の有無を確認し、無ければインストール方法を報告して停止）

yt-dlp --no-update --write-auto-subs --write-subs --sub-lang ja --sub-format vtt --skip-download -o "%(id)s" "{{YOU_TUBE_URL}}"

- 生成された .vtt（例: VIDEO_ID.ja.vtt / VIDEO_ID.ja.vtt 系）を読む
- 取得失敗時は推測で要約せず、エラー内容と利用可能な字幕言語の確認結果を報告して停止する
- 可能なら動画タイトル・尺・チャンネル名も取得してヘッダに使う（任意）

================================================================================
【Phase A】前処理（clean / enrich 相当。内部処理のみ。途中出力しない）
================================================================================

1. 無視する:
   - タイムスタンプ
   - [音楽][拍手] などのタグ
   - 同一フレーズの連続重複
   - チャンネル登録・宣伝の冗長部

2. 明らかな ASR 誤変換は文脈で修正する（例）:
   - 暴空→防空 / 石側→西側 / 石外線→赤外線
   - ペトリオット→パトリオット / ゼレンツキー→ゼレンスキー
   - 衛生・野生→衛星（文脈次第）
   - 英日混在（例: イェラブuga）は正規の片方に統一
   確信が持てない語は原文のまま残し、要約で強調しない

3. 内部メモを作る（最終出力には出さない）:
   - outline_hints: 動画が明示した章立て・「○つの柱」・テーマ見出し
   - entities: 組織・製品・人名とその属性（「誰の何」を混同しないため）
   - numeric_facts: 数値・距離・年・割合など字幕に出た数字
   - uncertain: 年号や数字が欠けていそうな箇所・意味不明な固有名詞

4. 長い動画は約10分単位で区切ってから要約する（コンテキスト切れ防止）

================================================================================
【Phase B】要約ルール（厳守）
================================================================================

- 事実は字幕（と上記内部メモ）のみに基づく
- 字幕にない年号・数値・固有名詞・因果関係は書かない
- uncertain は省略するか「不明」と書く（推測補完しない）
- 別組織・別製品・別人の属性を混ぜない
- 数値・固有名詞は字幕表記を優先し、英日混在表記を作らない
- 宣伝・おまけトークは最後のみ簡潔化、または省略
- 「字幕によると」「セグメントと連続」などのメタ記述は入れない
- 考え順: glossary（頭の中）→ 結論 → 章立て要約 → セグメント要約
- 再要約ループはせず、1回で完結する
