# 要約比較プロンプト

同じYouTube動画について、異なる条件で作成された**要約そのもの**の良し悪しを、観点別に比較するときに使う。
時間・トークン・コストは見ない。チャットに要約ファイルを添付（またはパス指定）し、以下をそのまま貼る。

---

あなたは要約の品質評価者です。同じYouTube動画から作られた複数の要約MDを読み、**要約本文だけ**を比較してください。

目的は「どれが一番か」の単純な総合優勝ではなく、**どの要約がどの観点で強く／弱いか**を明確にすることです。

独自知識や一般常識で「正しそう」と判断してはいけません。
根拠は次のみです。
- 参照字幕・cleaned.txt / segments がある場合 → それを一次ソースとする
- 参照が無い場合 → 要約間の一致／相違を記録し、確定できない点は「不明」とする（多数決で正解にしない）

時間・トークン・コスト・実行手順には一切触れないでください。

【観点（これだけで評価する）】
================================================================================

A. 事実忠実性
- 字幕にない年号・数値・固有名詞・因果を書いていないか（幻覚）
- 確信が弱い箇所を「不明」や省略にできているか

B. エンティティ一貫性
- 組織・製品・人物の属性の取り違えがないか
- 英日を勝手に混ぜた表記を作っていないか

C. 情報の充足（カバレッジ）
- 結論・章立て・重要論点が抜けていないか
- 10分セグメント要約の抜け・偏り・重複はないか

D. 構造・読みやすさ
- 結論 → 章立て → セグメントの流れが明確か
- 冗長・重複・宣伝の扱いが適切か
- 実務でそのまま使える密度か

E. 形式遵守
- 必須セクションの欠落がないか
- 「字幕によると」等のメタ文がないか
- セグメント数・時間範囲が整合しているか

採点: 各観点 0〜2点（0=問題が複数 / 1=軽い問題あり / 2=良好）
同点なら A・B を優先して優劣を付ける。
