# ASR 補正方針（事前辞書は使わない）

**どんな動画でも URL 一発で動かす。** ジャンル別の変換表を事前に用意・メンテしない。

スクリプトは表記ゆれの固有名詞置換を**しない**（ノイズタグ除去と重複除去のみ）。
意味の補正は、エージェントが**その動画の字幕文脈だけ**で、要約時に一度だけ行う。

## その場での補正ルール

1. 同一動画内の繰り返し・周辺語・タイトル／チャンネル情報から、明らかな誤認識だけ直す
2. 確信が持てない語は字幕のまま残す。要約でも強調しない（`uncertain`）
3. 外部知識や「一般的に正しい表記」で埋めない（事実の補完と混同しない）
4. 英日が1語の中で混ざっている場合だけ、字幕全体で優勢な表記に寄せる
5. 同一物のゆれ（例: A社／エー社）は、字幕内で多い方か最初の明示形に統一する

## 補正してよい／いけない

| してよい | いけない |
|---|---|
| 直後の文で意味が一意になる誤変換 | 年号・数値・因果の補完 |
| 同じ固有名詞が別箇所で正しく出ているときの合わせ込み | 動画に出ない別名への置換 |
| 明らかに壊れた助詞・同音漢字の一意な直し | 「たぶんこの製品だろう」推測 |

## 例（パターン説明のみ。辞書として載せない・機械適用しない）

- 防衛の話で「暴空」→「防空」（周辺にレーダー・迎撃がある）
- 偵察の話で「衛生」→「衛星」（画像・軌道の文脈）
- 人名が直後に正しく繰り返される → 正しい方に寄せる

これらは軍事動画専用ルールではない。**当該字幕の一貫性**で判断する。
