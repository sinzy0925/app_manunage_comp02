# 出力前チェック（詳細）

要約MDを書く直前に確認する。

## 必須

- [ ] `meta.json` の `ok` が true。false なら要約していない
- [ ] `字幕ソース` が「日本語手動」または「日本語自動生成」
- [ ] セグメント見出し数が `meta.json` の `segment_count` と一致
- [ ] 各セグメントの時間範囲が `segments/seg_XX.txt` 先頭メタと一致
- [ ] 年号・距離・割合・固有名詞は字幕／内部メモ由来のみ
- [ ] `uncertain` 由来の欠落を推測で埋めていない（省略 or 「不明」）
- [ ] 組織Aの属性を組織Bに付けていない
- [ ] 製品・人物・組織の属性混同がない
- [ ] 英日混在表記を新規に作っていない
- [ ] 「字幕によると」「前のセグメントでは」等のメタ文がない
- [ ] 宣伝・チャンネル登録誘導は末尾簡潔 or 省略
- [ ] チャットに開始時刻・終了時刻・所要時間を出した
- [ ] MD本体に pipeline ログや Phase A 生メモを混ぜていない
- [ ] ジャンル別の事前辞書や追加字幕を要求していない（URL一発）

## 失敗時の報告テンプレ

```text
字幕取得に失敗したため要約を中止しました。

- URL: …
- エラー: …
- 利用可能な字幕言語: …
- yt-dlp 導入状況: …
```
